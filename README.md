## � Resumo Rápido (versão compacta)

Uma plataforma PWA focada em empoderar mães com acessibilidade, comunidade e recursos práticos. Principais pontos em 3 linhas:

- Assistente de voz otimizada (PT-BR) e acessibilidade para PCDs
- Módulos educacionais, centro de documentos e chat comunitário moderado
- Sistema de emergência com envio de localização e contatos — backend planejado para funcionalidades avançadas

Leia a versão completa abaixo; várias seções grandes estão dobráveis para leitura rápida.

## 🚀 **Visão Geral do Projeto**

O **Mãe Conecta** é uma **plataforma revolucionária** que transforma burocracia em ação, oferecendo suporte real e prático para mães e cuidadores. Mais que um app - é uma **ferramenta de empoderamento social**.

### 🎯 **Problema Identificado**
- **2.3 milhões** de crianças com deficiência no Brasil
- **89%** das mães enfrentam dificuldades burocráticas
- **67%** não sabem quais direitos possuem
- **78%** sentem-se isoladas e sem apoio

### 💡 **Nossa Solução**
Uma plataforma que **elimina barreiras** e **conecta mães** aos seus direitos, oportunidades e comunidade através de:

## 🚀 **Como Executar**

### **Quick Start (2 minutos)**
```bash
# Clone o repositório
git clone https://github.com/MarcioGil/Mae-Conecta.git
cd Mae-Conecta

# Instale dependências
cd frontend && npm install

# Execute em desenvolvimento
npm run dev

# Acesse http://localhost:3000 🎉
```

### **Testando Recursos Únicos**

#### 🎙️ **Assistente Clara de Voz**
1. Clique no **botão flutuante** (microfone)
2. Diga: *"Meus direitos"* ou *"Ajuda"*
3. Ouça a resposta em voz natural

#### ♿ **Acessibilidade Total**
1. Clique na **engrenagem** (canto superior direito)
2. Teste: **Alto contraste**, **Fonte grande**, **Modo escuro**
3. Use **Tab** para navegação por teclado

#### 📱 **PWA Installation**
1. Acesse via **Chrome mobile**
2. Toque em **"Adicionar à tela inicial"**
3. Use como **app nativo**
